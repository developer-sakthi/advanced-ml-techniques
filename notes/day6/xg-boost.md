# XG Boost

- It is an advanced and optimized version of boosting that is designed to be faster , more efficient , more accurate
- XGBoost became extremely popular in machine learning competitions and real-world applications because of its strong performance.
- Many winning solutions in data science competitions use XGBoost.

### Key idea

- Instead of just increasing importance of misclassified samples like AdaBoost, XGBoost focuses on learning the errors of the previous models.

- Each new model tries to correct the remaining errors.

- Gradually, the combined models become very accurate.

- XGBoost also includes built-in techniques that help prevent overfitting.

### Why XGBoost is powerful

- XGBoost includes several improvements:

- Regularization to prevent overfitting

- Efficient memory usage

- Parallel processing

- Automatic handling of missing values

- Built-in feature importance analysis
